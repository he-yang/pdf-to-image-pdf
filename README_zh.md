# PDF-to-Image-PDF

一款功能强大的开源 PDF 处理工具，纯本地运行，不上传任何文件到互联网，保护隐私安全。

[English README](README.md)

## ✨ 功能特性

### 📄 PDF 转换
- 将 PDF 文件转换为图片（PNG 格式）
- 将 PDF 文件转换为图片版 PDF（每页一张图片）
- 支持批量处理多个 PDF 文件
- 可添加自定义水印（文字、位置、大小、透明度）
- 支持多种 DPI 设置（72/150/300/600）

### 📑 页面提取
- 从 PDF 文件中提取指定页码范围的页面
- 支持多种输入格式：
  - 单个页码：`5`
  - 连续范围：`1-5`
  - 多个范围：`1-5, 8, 10-15`
- 保持原文件格式和质量

### 📚 PDF 合并
- 将多个 PDF 文件合并为一个文件
- 支持调整文件合并顺序
- 保留书签和元数据

### 🔒 隐私保护
- 所有处理均在本地完成，不上传任何文件
- 不收集任何用户数据
- 完全免费，无广告

## 📝 使用说明

### PDF 转换
1. 点击「选择文件」按钮，选择一个或多个 PDF 文件
2. 选择输出格式（图片版 PDF 或仅图片）
3. 可选：启用水印并设置水印参数
4. 选择 DPI 质量
5. 点击「开始转换」按钮

### 页面提取
1. 切换到「提取页面」标签
2. 点击「选择 PDF 文件」按钮，选择一个 PDF 文件
3. 在输入框中输入页码范围（如 `1-5, 8`）
4. 点击「提取页面」按钮

### PDF 合并
1. 切换到「合并 PDF」标签
2. 点击「添加 PDF 文件」按钮，选择多个 PDF 文件
3. 调整文件顺序（如有需要）
4. 点击「合并 PDF」按钮
